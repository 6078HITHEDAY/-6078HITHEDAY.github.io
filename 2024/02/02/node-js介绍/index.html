Node.js

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时环境，它可以让 JavaScript 代码在服务器端执行，从而实现高性能的网络应用开发。

特点

  • 事件驱动：Node.js 采用事件循环机制，使得 I/O 操作非阻塞，提高了并发处理能力。
  • 单线程：Node.js 只有一个主线程，避免了多线程编程的复杂性，但也意味着无法充分利用多核 CPU。
  • 模块化：Node.js 使用 CommonJS 规范定义模块，通过 require 函数加载模块，通过 exports 对象导出模块。
  • 跨平台：Node.js 可以在 Windows, Linux, macOS 等多种操作系统上运行。
